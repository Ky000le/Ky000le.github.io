建博客二三事

发表于 2018-03-02 | 分类于 博客

自建博客体验还算可以，但是我对此还有些别的看法。

ssh步骤出现提示无此目录。我的解决方法是暂且搁置，待rsa密钥生成后一切正常，这种体验不是很好。

npm安装hexo时差点卡死在这个步骤。换淘宝源解决了。

期待 yuwriter 作者更新发布博客功能，这应该不算是很难的事情，每次手动cmd输入hexo d -g很麻烦的。
